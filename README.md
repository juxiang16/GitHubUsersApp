GitHub Users App

アプリ概要：
GitHub REST APIを利用してGitHubユーザーの一覧および詳細情報を表示するAndroidアプリです。

使用技術：
Kotlin
Jetpack Compose
Retrofit
Gson
Coil
Navigation Compose
Coroutines
Git / GitHub

主な機能：
GitHubユーザー一覧の取得・表示
ユーザー詳細情報の取得・表示
プロフィール画像表示
画面遷移（Navigation）

使用API：
GET /users
GET /users/{login}

開発環境
Android Studio
Kotlin
Git
GitHub
